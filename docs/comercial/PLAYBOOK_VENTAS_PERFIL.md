# Playbook de Ventas por Perfil y Nivel

## Objetivo
Estandarizar discurso comercial, manejo de objeciones y reglas de negociacion sin romper el margen minimo.

## Mensaje por perfil
1. Docente: ahorro de tiempo, menos friccion operativa y entrada paga accesible.
2. Coordinacion: estandarizacion, control de proceso y menor reproceso.
3. Institucional: cumplimiento, continuidad, SLA y riesgo reducido.
4. Socio de Canal: salida rapida al mercado y monetizacion de cartera.

## Objeciones comunes y respuesta marco
1. "Esta caro para empezar"
- Respuesta: iniciar en nivel Esencial/Operador con alcance controlado y ascenso por valor probado.

2. "Necesito todo incluido desde el inicio"
- Respuesta: separar producto base y complementos de servicio para pagar solo valor necesario.

3. "Otro proveedor cobra menos"
- Respuesta: comparar costo total operativo y riesgo, no solo ticket mensual.

4. "Necesito descuento mayor"
- Respuesta: descuentos dentro de politica, nunca por debajo de margen minimo.

## Reglas de negociacion
1. No comprometer descuento que rompa margen bruto >= 60%.
2. Priorizar concesiones de plazo/alcance antes que reduccion fuerte de precio.
3. Descuentos excepcionales requieren comite comercial-financiero.
4. Todo descuento debe registrar razon, impacto esperado y plan de expansion.

## Matriz de descuentos permitidos (referencial)
1. Niveles de entrada (Esencial/Operador): 0%-10%.
2. Niveles intermedios (Impulso/Control/Gestion/Crecimiento): 0%-15%.
3. Niveles avanzados (Rendimiento/Acompanamiento/Integral/Multisede/Marca Blanca): 0%-20% con condicion anual.
4. >20%: solo con aprobacion formal y caso documentado.

## Criterios de avance de etapa comercial
1. Discovery completo por perfil.
2. KPI de piloto acordados.
3. Propuesta con nivel recomendado y alternativa superior.
4. Validacion de margen y payback antes de enviar oferta final.
