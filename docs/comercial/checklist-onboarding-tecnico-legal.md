# Checklist Onboarding Tecnico-Legal

## Tecnico
- [ ] Stack local funcional (`/api/salud/live`).
- [ ] Portal cloud funcional (`/api/portal/salud/live`).
- [ ] Roles/permisos validados.
- [ ] Flujo OMR end-to-end validado.

## Seguridad
- [ ] Secretos cargados por canal seguro.
- [ ] CORS y rate limit ajustados.
- [ ] Logs con requestId y sin PII.

## Legal/privacidad
- [ ] Aviso integral y corto entregados.
- [ ] Politica de retencion validada.
- [ ] Canal ARCO comunicado.
- [ ] Responsable de privacidad identificado.

## Comercial
- [ ] Alcance demo firmado.
- [ ] KPIs y criterios de exito acordados.
- [ ] Fecha de comite de cierre definida.
