# Checklist de Aprobacion de Precio por Nivel

## Datos minimos del caso
- [ ] Persona objetivo definida.
- [ ] Nivel propuesto definido.
- [ ] Pais/mercado objetivo definido.
- [ ] Modalidad de pago definida (mensual/anual).

## Validacion financiera
- [ ] Margen bruto estimado >= 60%.
- [ ] LTV/CAC estimado >= objetivo de la linea.
- [ ] Payback CAC <= objetivo de la linea.
- [ ] Costo de servicio (soporte/onboarding/compliance) calculado.

## Validacion comercial
- [ ] Propuesta incluye ruta de upsell clara.
- [ ] Limites del nivel de entrada documentados (si aplica).
- [ ] Descuento dentro de politica vigente.
- [ ] Si descuento excepcional: aprobacion de comite adjunta.

## Validacion operativa y legal
- [ ] SLA y alcance de soporte documentados.
- [ ] Condiciones de prueba promocional alineadas a privacidad/ARCO.
- [ ] Riesgos operativos identificados y mitigados.

## Decision final
- [ ] Precio aprobado.
- [ ] Precio condicionado (especificar condiciones).
- [ ] Precio rechazado (especificar motivo).
