# Licenciamiento Comercial y Niveles

## Esquema Base por Edicion
- **Edicion Comunitaria (AGPL)**: uso y modificacion del nucleo abierto para adopcion inicial.
- **Edicion Comercial**: uso propietario + automatizacion y soporte para equipos docentes.
- **Edicion Institucional**: cobertura completa, cumplimiento avanzado y operacion institucional.

## Glosario Comercial Canonico
- **Hoja de Ruta**
- **Nivel**
- **Tipo de Licencia**

## Portafolio de Niveles por Perfil de Usuario
### Linea Docente Individual
1. **Docente Esencial (Producto)**
   Nivel de entrada pago de bajo costo, por encima de la edicion gratuita.
   Incluye: base comercial para 1 docente, incorporacion rapida y soporte estandar.
2. **Docente Impulso (Producto)**
   Para mayor demanda operativa y necesidad de productividad.
   Incluye: Esencial + automatizacion adicional y reportes clave.
3. **Docente Rendimiento (Producto)**
   Para docente con carga alta y seguimiento continuo.
   Incluye: Impulso + analitica ampliada y control operativo reforzado.
4. **Docente Acompanamiento (Servicio)**
   Para docente que requiere soporte cercano.
   Incluye: Rendimiento + acompanamiento prioritario y sesiones guiadas.

### Linea Coordinacion Academica
5. **Coordinacion Esencial (Producto)**
   Para coordinaciones pequenas con estandarizacion inicial.
   Incluye: control base para equipo reducido.
6. **Coordinacion Control (Producto)**
   Para coordinaciones con necesidad de seguimiento por periodos.
   Incluye: Esencial + tableros operativos y trazabilidad extendida.
7. **Coordinacion Gestion (Producto)**
   Para operacion recurrente con mayor volumen.
   Incluye: Control + integraciones prioritarias y productividad avanzada.
8. **Coordinacion Acompanamiento (Servicio)**
   Para equipos que requieren operacion asistida.
   Incluye: Gestion + mesa de ayuda prioritaria y adopcion guiada.

### Linea Institucional
9. **Institucional Esencial (Producto + Servicio)**
   Para universidades con base formal de seguridad y cumplimiento.
   Incluye: SLA base, controles operativos y gobierno minimo.
10. **Institucional Integral (Producto + Servicio)**
    Para instituciones con continuidad operativa exigente.
    Incluye: Esencial + endurecimiento ampliado y soporte institucional.
11. **Institucional Multisede (Producto + Servicio)**
    Para operacion por campus o grupo de sedes.
    Incluye: Integral + administracion centralizada y politicas compartidas.
12. **Institucional Sector Publico (Anexo)**
    Para sujetos obligados del sector publico.
    Incluye: Multisede/Integral + paquete documental y operativo de cumplimiento.

### Linea Socio de Canal
13. **Socio Operador**
    Para socio que opera EvaluaPro para terceros en alcance acotado.
14. **Socio Crecimiento**
    Para socio con cartera en expansion y necesidad de escalamiento.
15. **Socio Marca Blanca**
    Para comercializacion con identidad del socio bajo contrato.
16. **Socio OEM Integrado**
    Para integracion profunda en oferta de tercero con condiciones OEM.

## Principios de Precio y Rentabilidad
- Cada linea debe tener al menos 1 nivel de entrada pago de bajo costo.
- Ningun nivel comercial se vende por debajo del piso de margen objetivo.
- El precio se calcula con base en:
  - valor de negocio entregado,
  - costo de servir (infra, soporte, incorporacion, cumplimiento),
  - riesgo operativo y legal,
  - potencial de expansion de cuenta.
- Modelo mixto recomendado:
  - componente de producto (suscripcion),
  - componente de servicio (SLA, soporte, acompanamiento, cumplimiento).
- KPI minimos para aprobar precio por nivel:
  - margen bruto positivo y sostenible,
  - LTV/CAC saludable,
  - payback de CAC dentro del objetivo comercial.

## Promocion de Prueba Comercial
- **Periodo de prueba recomendado**: 35 dias.
- **Oferta de entrada recomendada al finalizar**: nivel de entrada del perfil correspondiente (Docente Esencial, Coordinacion Esencial, Institucional Esencial, Socio Operador).
- **Intercambio de valor de la promocion**:
  - el cliente recibe acceso temporal a capacidades comerciales,
  - EvaluaPro recibe datos de uso y operacion para personalizar incorporacion, soporte y propuesta comercial.
- **Condiciones minimas**:
  - consentimiento informado y documentado,
  - finalidades explicitadas (producto, soporte, conversion comercial),
  - retencion y ejercicio ARCO/DSR conforme politica vigente.

## Modalidades de Pago
- **Mensual**: menor barrera de entrada.
- **Semestral**: balance costo-flexibilidad.
- **Anual**: mejor costo total para operacion continua.
- **Unico pago**: recomendable cuando se busca estabilidad presupuestal.

## Recomendacion de Actualizaciones (Pago Unico)
- Ventana sugerida: **12 a 24 meses** de actualizaciones incluidas.
- Al terminar la ventana: plan de mantenimiento anual opcional.

## Politica de Hoja de Ruta por Tipo de Cliente
- **Edicion Comunitaria**: hoja de ruta publica sin compromiso de fecha.
- **Comercial estandar**: prioridad por valor de negocio y criticidad operativa.
- **Institucional / Multisede / Socio de Canal / OEM**: prioridades acordadas por contrato y comite de seguimiento.

## Nota de Rentabilizacion Comunitaria
- La Edicion Comunitaria se rentabiliza por conversion a niveles de entrada, servicios complementarios, soporte premium opcional y canal de socios.
